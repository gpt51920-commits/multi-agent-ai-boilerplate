# 🚀 Multi-Agent AI SaaS Boilerplate

Повноцінний шаблон для створення SaaS-продуктів на базі мультиагентних AI-систем. Побудований на **FastAPI**, **LangGraph**, **Groq (Llama 3)** та **LangFuse** для трасування.

## ✨ Можливості
- 🧠 **Research → Write → Review Agent**: Автоматизований цикл створення контенту з самоперевіркою.
- 🎧 **Customer Support Router**: Класифікація запитів, визначення пріоритету та автоматична ескалація.
- 📊 **LangFuse Integration**: Повне трасування викликів, витрати токенів та візуалізація workflow.
- ⚡ **Blazing Fast**: Використання Groq API для мілісекундних відповідей Llama 3.
- 🐍 **Production-Ready**: Асинхронний код, віртуальне оточення, типізація Pydantic.
